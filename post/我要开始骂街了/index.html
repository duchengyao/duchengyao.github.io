我要开始骂街了

发表于 2024-01-12 分类于 其它笔记 阅读次数：

新人培训

上家公司是个国企，刚入职的时候有一周新人培训。

有天作业是小组做PPT，讲科学家的故事，讲完后全员投票，排名越高，得的小红花印章越多。

其他组有行政，销售，积极活跃，之前已经积累了很多小红花；

我们组除了大懒蛋就是程序员，非必要不发言，第三天了才只有两朵。

我假装不会做 ppt 随便百度了几张图丢群里，我们组其它的大懒蛋也类似；

最后好不容易抽签选了个大冤种整合 ppt 和做演讲。

演讲那天，我们组倒数第二个上台。看着其他组要么激情澎湃，要么方式新颖，甚至还有演话剧来呈现科学家精神的。

我们组只有3分钟，算上 “标题” 和 “Thanks” 5页ppt。

我们问老师，能不能弃权不参加比赛了，太丢人了。老师不同意。

我们演讲的同事一路磕磕绊绊，把3页 ppt 讲了10 分钟。

全部讲完以后，每个人用小程序投票，不能投自己组。

最后投票结果我们组第一，票数断崖式领先第二名。

发言的那哥们甚至真以为自己演讲能力极强。

防御性编程

上个组有个老员工，特别擅长防御性编程（为了体现程序员们在公司代码中的不可替代性，防止被裁）。

a) 混乱的 git commit

我们有一部分代码是基于开源库修改的，那个老员工提交 git 的时候，把代码删了 commit 一次，更新开源库的 master+改新代码 commit 一次。

这样更新开源库的master+改的新代码放在一个commit里，几乎没人能看懂他改了啥。

b) fastdds

我 22 年有段时间的工作是把系统现有的消息总线替换成 fastdds。

原有的消息总线是领导年轻的时候写的（估计有个几十年了吧）。领导发现自己写的消息总线有瓶颈，并且几乎无法维护了，就让我改成行业比较优秀的开源项目。

领导已经不管代码了，旧的消息总线只有那个老员工能看懂。为了自己的不可替代性，他使出了各种手段阻挠新消息总线的部署。

我当时刚来这个组没多久，没想到改点代码还能牵扯到利益关系。

最开始，我消息总线的代码改好以后，提交merge请求，并催促review。

他一直找性能问题、稳定性问题，等各种借口拒绝合并，还到处散布fastdds有问题的谣言。

（直到现在，领导还觉得fastdds用在我们系统里有各种问题，但不知道是什么问题。）

后来 fastdds 终于合并了，但测试出一个bug，这个bug在我博客的文章[1,2]中有说明。

这个 bug 我在22年9月改好了。

但是每次外出实验，fastdds 都会崩溃。

每次实验复盘，那个老员工都会说杜承垚负责的消息总线不稳定，连不上。

但代码在实验室测试一切正常。

直到23年1月我发现外出实验那台电脑没更新我的代码。

我找他对峙，他承认。

c) 某次实验

有次实验，领导中午临时通知我提前一天去三亚，讨论问题，内容保密，先去了再说。（原定跟大部队第二天出发）

那天刚好我过生日，推掉了生日聚会，中午赶紧起床，去医院开足两周的药，订机票，凌晨4点到了酒店，第二天10点参加讨论。

讨论的时候，发现跟我完全没关系，我就回屋补觉了。

醒来以后，听说一个老员工造谣我先到三亚，是因为跟我老婆提前过去玩了几天。

还传我到这边不工作，来了只知道睡觉。

并且是传给领导，领导后来告诉我的。我找他对峙，他承认。

d) 后续

那个老员工还有其它一些损人不利己的事就不一一说了。

但最有意思的是，我把这些事告诉领导，领导竟然毫不在意。

并且领导的意思是，领导不看代码，整个项目是否能运行是那个老员工负责，老员工说什么就是什么。

我想说他的防御性编程做的确实值得学习。然后我就离开那个破组了。

一个长江博士

之前组里有个叫长江的博士。

当时我们项目的大领导离职了，空降来了一个代理领导。他来的目的就是把我们组拆散，把人恶心走，然后把资源回收给他自己的组。

代理领导一眼就看中了这个长江博士。

这个博士名字里面带了个长江，代理领导就叫他长江博士。我们同事一般都叫他狗博。

狗博感觉像是野鸡大学毕业的，刚毕业就来这当了领导，管了七八个人，自己觉得牛逼的不行。

成天向我们炫耀自己拿了华为的offer，还教我们怎么做人，怎么做事。

但其实软件都不会安装，也不会用。

我们组除了他以外，至少有两个发过顶刊、顶会的大佬。他给我们组半年定的目标是学会如何使用工具。

我问他具体什么目标，他说我们的目标是学会用工具看GPU的占用率。

我说这些我都会了，然后呢？他说不上来。

还给我们开PUA大会。还背着没来同事说他坏话。

那个同事刚好是我的好朋友。像我这种经历过PUA的反手一个录音，发给同事。

把我朋友气够呛，然后联合我们组员一起把这傻逼狗博给罢免了。

没多久他就离职了，去向不明。我都好奇他这水平竟然还有公司要他，也就华为愿意要了吧，一个应届生保底的公司。

录音我回头剪辑一下发出来。
